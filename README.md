# historyquizbot
# 🏛 History Quiz Telegram Mini App

Интерактивная ретро-викторина по историческим личностям, выполненная в стиле пиксельных аркадных игр.

Проект можно запускать как обычный сайт или использовать как Telegram Mini App через WebApp API.

---

## 🎮 Описание проекта

Это викторина, где игроку предлагается угадывать исторических деятелей по фактам.

Игрок проходит серию вопросов, выбирает правильный ответ и получает результат в конце.

---

## ⚙️ Основные функции

- 🎯 Викторина из 10+ вопросов
- ❤️ Система жизней (в зависимости от сложности)
- 🎚 Выбор уровня сложности (easy / medium / hard)
- 🧠 Историческая справка после каждого ответа
